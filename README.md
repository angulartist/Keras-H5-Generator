version number: 0.0.1

Overview
--------

A dead simple Keras HDF5 ImageDataGenerator

Installation / Usage
--------------------

To install use pip:

    $ pip install hdf5generator


Or clone the repo:

    $ git clone https://github.com/angulartist/hdf5generator.git
    $ python setup.py install
